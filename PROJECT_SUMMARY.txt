📦 PROJECT SUMMARY
══════════════════════════════════════════════════════════════════════════════

A fully functional web-based music sequencer inspired by classic tracker
software (FastTracker 2, ProTracker). Built with vanilla JavaScript, HTML5
Canvas, Web Audio API, and Web MIDI API.

✨ FEATURES IMPLEMENTED
══════════════════════════════════════════════════════════════════════════════

✅ Audio Features
   • Web Audio API synthesis engine
   • 3 independent monophonic tracks 
   • 16-step pattern sequencer
   • Square, Triangle, Sawtooth waveforms
   • ADSR envelope shaping
   • Master + per-track volume control
   • Tempo control (30-300 BPM)

✅ User Interface
   • Canvas-based piano roll grid
   • Dark FastTracker-inspired theme  
   • 2-octave note range (C4-G5, expandable)
   • Real-time playback indicator
   • Track selection buttons
   • Waveform selector
   • Volume sliders

✅ MIDI Features
   • Web MIDI API device selection
   • Simultaneous internal audio + MIDI output
   • Standard MIDI file export (.mid)
   • Multi-channel MIDI routing

✅ Control & Shortcuts
   • Keyboard shortcuts (Space, Delete, 1-3, Arrows)
   • Full mouse interaction
   • Play/Stop/Clear buttons
   • BPM adjustment

✅ Data Management
   • Browser localStorage pattern saving
   • JSON export/import
   • Event-driven state management
   • Serializable application state

✅ Architecture

🚀 HOW TO USE
══════════════════════════════════════════════════════════════════════════════

1️⃣  OPEN THE APPLICATION
    • Direct: Open index.html in browser
    • Or: Run local server (see QUICKSTART.md)

2️⃣  PLACE NOTES
    • Click grid cells to add notes
    • Each row = different pitch
    • Each column = time step
    • Click again to remove

3️⃣  PLAY
    • Press SPACE or click Play button
    • Adjust BPM with slider
    • Change waveform selector
    • Adjust volume

4️⃣  EXPORT
    • Save to browser: console command
    • Export MIDI: click button
    • Export JSON: console command
